Fact Check

환영합니다! 👋

Fact Check는 웹에서 발견한 정보의 신뢰성을 확인하는 데 도움을 주는 확장 프로그램입니다.

주요 기능

  • 📝
    텍스트 팩트 체크

    텍스트를 드래그하고 우클릭하여 사실 가능성을 확인하세요

  • 🖼️
    이미지 검증

    이미지를 우클릭하여 AI 생성 여부를 확인하세요

사용 방법

  1. 웹페이지에서 텍스트를 드래그하거나 이미지를 찾습니다
  2. 우클릭하여 "Fact Check" 메뉴를 선택합니다
  3. 팩트 체크 결과를 확인합니다
팩트 체크 활성화

확장 프로그램의 팩트 체크 기능을 켜거나 끌 수 있습니다

백그라운드 감지

페이지 방문 시 자동 팩트 체크 감지를 켜거나 끄세요

API Base URL

팩트 체크 API 서버 주소를 입력하세요
