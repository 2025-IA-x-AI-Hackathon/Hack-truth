HackTruth에 오신 것을 환영합니다! 👋

HackTruth는 웹에서 마주한 콘텐츠의 신뢰도를 빠르게 확인할 수 있는 팩트체크 도구입니다.

주요 기능

  • 📝
    텍스트 팩트 체크

    드래그한 텍스트를 우클릭해 사실 가능성을 확인합니다.

  • 🖼️
    이미지 팩트 체크

    이미지를 우클릭해 AI 생성 여부와 조작 징후를 분석합니다.

  • 🎬
    유튜브 영상/쇼츠 검증

    AI 생성 영상 여부와 거짓 주장 가능성을 판별합니다.

서비스 소개

유튜브 영상/쇼츠 검증 : AI 생성 영상 유무 및 거짓 주장 판별
백그라운드 감지 : 방문한 페이지의 거짓정보인 텍스트가 포함되어있는지 가능성을 판별합니다.

팩트 체크 활성화

확장 프로그램의 팩트 체크 기능을 켜거나 끌 수 있습니다

백그라운드 감지

페이지 방문 시 자동 팩트 체크 감지를 켜거나 끄세요
