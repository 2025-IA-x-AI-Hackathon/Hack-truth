
- 제목(Subject): . 50자 이내, 대문자로 시작, 명령문, 마침표 사용 금지
- 본문(Body): '무엇을', '왜' 변경했는지 설명, 필요시 여러 줄, 72자마다 줄바꿈
- 꼬리말(Footer): 관련 이슈, 트래커 등(선택)

## 타입(Type) 리스트

- **feat** : 새로운 기능 추가
- **fix** : 버그 수정
- **docs** : 문서 수정
- **style** : 코드 스타일, 포맷 변경 (동작 무관)
- **refactor** : 코드 리팩토링 (동작 무관)
- **test** : 테스트 코드 추가/수정/삭제
- **chore** : 기타 변경사항(빌드, 설정 등)
- **design** : UI/UX 변경
- **comment** : 주석 변경
- **init** : 프로젝트 초기 생성
- **rename** : 파일/폴더명 변경
- **remove** : 파일 삭제
